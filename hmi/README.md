# HMI Documentation

The private archive contains KTP700 Basic and WinCC Runtime Advanced targets. Original compiled scripts are excluded because they contain local paths and private metadata.

Public contents:

- [`screen-map.md`](screen-map.md): reconstructed operating areas;
- [`sanitized-scripts/`](sanitized-scripts/): newly authored VBScript examples based on recovered behavior.

Generic tag names in the examples must be mapped to reviewed PLC/HMI tags. They are not drop-in scripts.

# HMI Screen Map

| Area | Documented controls | Documented feedback | Unknowns to verify |
|---|---|---|---|
| Manual | axis direction/speed, direct movement | axis/sensor status | output arbitration and stop priority |
| Semi-automatic | X/Y entry and start | range error, target/actual position | exact limits/tolerance and units |
| Automatic | four numeric selections and sequence start | sequence progress | coordinate lookup and dosing quantities |
| Stepper | enable, up/down direction and three speeds | position/status | pulse frequencies and zeroing behavior |
| CSV/PC extension | file path/load and vector transfer | load/error/count status | exact screen objects and handshake |

The report says the PC runtime combines the four panel pages into one screen with four quadrants.

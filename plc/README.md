# PLC Documentation

This directory contains a public functional reconstruction, not an export of the original Siemens blocks.

- [`block-map.md`](block-map.md): recovered functional areas and evidence.
- [`io-map.md`](io-map.md): sanitised I/O map with conflicts.
- [`sequence-description.md`](sequence-description.md): mode and point-list sequence.
- [`sanitized-extracts/`](sanitized-extracts/): non-deployable pseudocode.

The private projects target TIA Portal V15. They must be opened, compiled and compared with the actual machine before any engineering use.

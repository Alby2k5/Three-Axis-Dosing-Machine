# Public Evidence Images

This directory contains two downscaled JPEG derivatives made from the supplied 2026 inspection photographs:

- `machine-overview-redacted.jpg` — overall Cartesian machine structure; the machine identification plate is masked.
- `control-cabinet-redacted.jpg` — cabinet architecture; a network-address label is masked.

## Processing applied

- resolution reduced for documentation;
- EXIF, GPS and device metadata removed;
- machine serial/address labels masked;
- no people are visible.

## Evidence boundary

The images establish visible presence and configuration only. They do not prove electrical continuity, correct wiring, safe operation, PLC configuration, calibration or successful functional testing.

## Rights and publication check

These image derivatives are not covered by the repository MIT licence unless the photograph rightsholder explicitly licenses them. Before publishing or redistributing the repository, confirm ownership/permission and re-inspect the full-resolution JPEGs for residual identifiers, reflections and third-party material. Remove this directory when permission is uncertain.
